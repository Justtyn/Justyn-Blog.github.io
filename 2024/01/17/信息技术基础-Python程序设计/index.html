第三章 基本数据类型

一、数字类型

  • 整数类型(1010)

    • 进制只是整数值的表示的形式，不同进制的整数之间直接运算和比较

    • 进制之间的运算结果默认以十进制方式展示

    • 4种进制表示方式

      十进制 无引导符号 整数：0、1、1010
      二进制 0b / 0B 由0-1组成：0b1010、0B1001
      八进制 0o / 0O 由0-7组成：0o7010、0O4040
      十六进制 0x / 0X 由0-9、a-f、A-F组成：0x1010、0X1010

  • 浮点数类型(10.10)

    • Python 中浮点数类型必须带有小数部分

    • 浮点数的两种表达方式：一般表示(十进制形式)、科学计数法

      • 例如：1010.1、-1010.11、1.01e3、-1.01E-3
      • 科学技术法使用字母 e 或 E 作为幂符号，含义为：e = a · 10^b

    • pow(x,y) 函数用来计算 x^y 的值

      • 整数的运算精度比浮点数更高

      • 范例：

    • round() 函数用来限定运算结果保留的位数

      • round(x,d) 是一个四舍五入函数，表示对 x 进行四舍五入操作，参数 d 指定保留的小数位数

      • 范例：

  • 复数类型(10 + 10j)

    • Python 中复数以 “a + bj” 的形式表示，a 为实数，b 为虚部

    • 注：当 b 等于 1 时，1 不可省略，即 1j 表示

    • 复数的实部、虚部都是浮点类型

    • 对于复数 z 可以用 z.real 和 z.imag 分别获得它的实部的虚部

    • 共轭复数(实部相等，虚部互为相反数)

      • 复数 (a + bj) 的共轭复数为 (a - bj)

第四章 组合数据类型

一、组合数据类型基本概念

  • Python 语言中常用的组合数据类型分为：

    • 集合类型

      • 是一个元素集合，元素之间无序，相同元素在集合中唯一存在
      • 是一个具体的数据类型名称

    • 序列类型

      • 是一个元素向量，元素之间存在先后关系，通过序号访问，元素之间不排他
      • 典型代表：字符串类型、列表类型、元组类型
      • 是一类数据类型的总称

    • 映射类型

      • 是 “键值对” 数据的组合，没个元素是一个键值对 (key,value)
      • 典型代表是字典类型
      • 是一类数据类型的总称

  • 序列类型概述

    • 序列类型是一维元素向量，元素之间存在先后关系，使用序号访问

    • 访问序列中某个特定值时，通过下标标出即可，注：序列下标从 0 开始，最后一位是 -1

    • 字符串类型可以看做是单一字符的有序组合，列表则是可以使用多种类型元素的序列类型

    • x in s 若 x 是 s 的元素，则 True，否则 False
      x not in s 若 x 不是 s 的元素，则 True，否则 False
      s + t 连接 s 和 t
      s * n 将序列 s 复制 n 次
      s[i] 索引第 i 个元素
      s[i:j] 切片，返回序列 s 第 i (包含) 到 j (不包含) 个元素的子序列
      s[i:j:k] 返回序列 s 第 i (包含) 到 j (不包含) 个元素以 k 为步长的子序列
      len(s) 序列 s 的元素个数
      min(s) 序列 s 的最小元素
      max(s) 序列 s 的最大元素
      s.index(x) 序列 s 中第一次出现元素 x 的位置
      s.count(x) 序列 s 中出现元素 x 的次数

  • 映射类型概述

    • 映射类型每个元素是一个键值对，元素之间是无序的

    • 在序列类型中使用从 0 开始的正向递增序号进行索引

    • 而映射类型由用户自定义序号，即键

二、列表

  • 列表的定义

    • 列表是包含 0 个或多个元素的有序序列，可以对列表进行增、删、改、查

    • 无长度限制，元素类型无限制，不需要预定义长度

    • 使用 “[]” 表示，可以通过 list(x) 函数将集合或字符串替换成列表，list() 可以生成空数列

  • 列表的索引

    • 列表的索引用于获取列表中的一个元素，索引方式与其他序列类型一致

    • 索引序号不能超过列表的元素范围，否则会报 IndexError 错误

  • 列表的切片

    • 列表的切片用于获取列表的一个片段，即获取 0 个或多个元素，切片后返回仍是列表类型

    • 两种切片形式：ls[i:j]、ls[i:j:k]

    • 一般要求 i 小于 k，当大于时返回空列表

  • 列表的操作函数

    • ```python
      ls = [10, 1001, 1010, 10]

      print(len(ls)) # 4

      最大、最小函数要求列表个元素类型可以比较，否则会报错

      print(min(ls)) # 10

      lt = [“1010”, “11”, “Python”, “Hello”]

      比较字符串时，返回最大、最小长度的元素

      print(max(lt)) # ‘Python’

    • ls.append(x) 在列表最后增加一个元素 x
