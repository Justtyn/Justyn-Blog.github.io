第二章 Python 语言基本语法元素

一、程序的格式框架

  • 缩进

    • 使用 4个空格 / tab键 进行缩进（不能混合使用）

    • Python 对语句之间的层次关系没有限制，可以嵌套使用多层缩进

    • 缩进错误：”SyntaxError: unexpected indent“

    • 范例：

  • 注释

    • Python 使用 “#“ 表示一行注释的开始

    • 注释会被编译器或解释器过滤掉，不被执行

    • 三引号多行注释是一对连续的单引号(‘’’)或双引号(“””)

    • 同时三引号也用于字符串跨多行(此时字符串中可以包含换行符、制表符以及其他特殊字符)

    • 注释一般用于标注作者、版权信息、解释代码等

    • 范例：

  • 续行符

    • Python 使用反斜杠 “\” 表示续行

    • 续行符后必须立即换行

    • 范例：

二、语法元素的名称

  • 变量

    • 变量的值通过 “=” 赋值方式修改

    • 变量可以随时命名、随时赋值、随时使用

    • 范例：

  • 命名规则

    • 不能使用保留字

    • 变量名的首字符不能是数字

    • 变量名对大小写敏感

    • 除了下划线 ”_“ 之外，不能使用其他特殊字符

    • 范例：

  • 保留字

    is lambda None nonlocal not or pass
    raise return True try while with Yield

三、数据类型概述

  • 基本数据类型：数字类型(整数、浮点数、复数)、字符串类型

  • 组合数据类型：元组类型、集合类型、列表类型、字典类型

四、程序的语句元素

  • 表达式

    • 由数据和操作符组成，类似于数学的计算公式

  • 赋值语句

    • 由 “=” 表示赋值

      • 计算出等号右侧表达式的值，赋予左侧变量新的值
      • 等号右侧的数据类型同时作用于左侧的变量，即赋予变量新的数据类型

    • 元组赋值语句

      • 可以同时为多个变量赋值

      • 同时计算等号右侧所有表达式的值，并一次性给等号左侧对应变量赋值

      • 范例：

    • 多目标赋值语句

      • 将等号右侧的表达式的值同时赋值给多个等号左侧的变量，即一个数据由多个变量共享

      • 范例：

      • 在其他语言中需要引入一个中间变量以实现数值互换

      • 在Python中，利用元组赋值语句，无需中间变量即可实现数值互换

五、IPO与输入输出函数

  • IPO：input、process、output

    • Input：文件输入、网络输入、控制台输入、交互界面输入、随机数据输、内部参数输入
    • Process：”算法” - 程序的灵魂
    • Output：控制台输出、图形输出、文件输出、网络输出、操作系统内部变量输出

  • input() 函数 (输入)

    • 用于从控制台获得用户的一行输入，无论输入什么内容，都始终以字符串类型返回

    • 范例：

    • 与 input() 函数一起使用时，用于获取用户输入的数字
