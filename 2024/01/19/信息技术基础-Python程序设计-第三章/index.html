第三章 基本数据类型

一、数字类型

  • 整数类型(1010)

    • 进制只是整数值的表示的形式，不同进制的整数之间直接运算和比较

    • 进制之间的运算结果默认以十进制方式展示

    • 4种进制表示方式

      十进制 无引导符号 整数：0、1、1010
      二进制 0b / 0B 由0-1组成：0b1010、0B1001
      八进制 0o / 0O 由0-7组成：0o7010、0O4040
      十六进制 0x / 0X 由0-9、a-f、A-F组成：0x1010、0X1010

  • 浮点数类型(10.10)

    • Python 中浮点数类型必须带有小数部分

    • 浮点数的两种表达方式：一般表示(十进制形式)、科学计数法

      • 例如：1010.1、-1010.11、1.01e3、-1.01E-3
      • 科学技术法使用字母 e 或 E 作为幂符号，含义为：e = a · 10^b

    • pow(x,y) 函数用来计算 x^y 的值

      • 整数的运算精度比浮点数更高

      • 范例：

    • round() 函数用来限定运算结果保留的位数

      • round(x,d) 是一个四舍五入函数，表示对 x 进行四舍五入操作，参数 d 指定保留的小数位数

      • 范例：

  • 复数类型(10 + 10j)

    • Python 中复数以 “a + bj” 的形式表示，a 为实数，b 为虚部

    • 注：当 b 等于 1 时，1 不可省略，即 1j 表示

    • 复数的实部、虚部都是浮点类型

    • 对于复数 z 可以用 z.real 和 z.imag 分别获得它的实部的虚部

    • 共轭复数(实部相等，虚部互为相反数)

      • 复数 (a + bj) 的共轭复数为 (a - bj)

    • 范例：

二、数字类型的运算

  • 数值运算符

    • 算数运算符

      和：x + y 差：x - y 积：x * y
      商：x / y (浮点数类型) 整数除：x // y (不大于 x 与 y 之商的最大整数) 模：x % y (商的余数)
      幂：x ** y (x^y) 本身：+ x 负值：- x

      • 数值运算可能会改变结果的数据类型规则：

        • 整数与浮点数混合运算，结果为浮点数

        • 整数之间运算，除法 “/” 结果为浮点数

        • 整数或浮点数与复数运算，结果为复数

        • 范例：

    • 赋值运算符

      • 所有二元运算操作符 (+、-、*、/、//、%、**) 都可以与赋值符号(=)相连**

        形成增强赋值操作符 (+=、-=、 =、/=、//=、%=、*=)

      • 范例：

    • 比较运算符

      • 运算符 描述 实例
        x == y 比较两个对象是否相等 ( 1 == 2 ) –> False
        x != y 比较两个对象是否不相等 ( 1 != 2 ) –> True
        x > y 返回 x 是否大于 y ( 1 > 2 ) –> False
        x < y 返回 x 是否小于 y ( 1 < 2 ) –> True
        x >= y 返回 x 是否大于等于 y ( 1 >= 2 ) –> False
        x <= y 返回 x 是否小于等于 y ( 2 <= 2 ) –> True

    • 位运算符

      • 运算符 描述 实例
        & 按位与运算符
        | 按位或运算符

        << 左移运算符

      • 运算符 表达式 描述
        and x not y 二者都为 True 才会输出 True，否则都为 False
        or x or y 二者都为 False 才会输出 False，否则都为 True
        not not x 取反

        真值表

      • x y and not or
        True True True True False
        True False False True
        False True False True True
        False False False False

    • 成员运算符

      • 运算符 描述
        x in y 在指定序列 y 中找到 x 则返回 True，否则返回 False
        x not in y 在指定序列 y 中找到 x 则返回 False，否则返回 True

      • 运算符 描述
        x is y 判断二者是否引用自同一个对象，类似于 id(x) == id(y)
        x is not y 判断二者是否引用自不同对象，类似于 id(x) != id(y)

        范例：

    • 运算符优先级

      • 运算符 (由高到低) 描述
        ** 幂 / 指数运算符
        ~ 按位取反运算符
        *、/、%、// 乘、除、取模、取整
        +、- 加、减
        >>、<< 左移运算符、右移运算符
        & 按位与运算符
        |、^ 按位或运算符、按位异或运算符
        <、>、<=、>= 比较运算符
        ==、!= 等于运算符
        =、%=、/=、//=、-=、+=、*=、**= 赋值运算符
        is、is not 身份运算符
        in、not in 成员运算符
        not、and、or 逻辑运算符

  • 数值运算函数

    • abs(x) x 的绝对值
      divmod(x,y) 一个包含除数与余数的元组 (x // y,x % y)
      pow(x,y) / pow(x,y,z) 幂运算 (x ** y) / (x ** y % z)
      round(x) / round(x,d) 对 x 四舍五入，保留 d 位小数，缺省 d 则返回四舍五入整数
      max(a,b,…,z) 返回任意数量值的最大值
      min(a,b,…,z) 返回任意数量值的最小值

      复数 z = a + bj 的绝对值 abs(z) = √a^2 + b^2

三、字符串类型及格式化

  • 反斜杠字符( \ )

    • “ \ “ 是一个特殊字符，表示”转义”

    • 例如：换行：” \n “、反斜杠：” \ \ “、制表符(tab)：” \t “、单引号：” \‘ “、双引号：” \“ “

  • 字符串的索引

    • 作为字符序列，字符串可以对其中单个字符或字符片段进行索引

    • 内置函数 len() 可以获取字符串的长度

    • 字符串的正向递增索引从 0 开始，最后一个字符序号为 len(s) - 1

    • 字符串的反向递减索引从 -1 开始，最后一个字符序号为 -len(s)

    • 范例：

  • 字符串的切片

    • 对字符串中某个子串或区间的检索称为切片 - a[M:N]

    • 切片获取字符串从 N 到 M (不包含 M )的子字符串

    • 切片要求 N 和 M 都在字符串索引区间内，如果 N 大于 M 则返回空字符串

    • N 缺失时，默认 N 为 0，从字符串第一个字符开始

    • M 缺失时，默认表示到字符串结尾

    • 范例：

  • format() 方法的基本使用

    • 字符串整合变量时需要使用字符串的格式化方法，用于解决字符串和变量同时输出的格式格式安排问题

    • 范例：

  • format() 方法的格式控制

    • 格式控制语法格式 { <参数序号> : <格式控制标记符> }

      ： 引导符号
      <填充> 用于填充的单个字符
      <对齐> < 靠左对齐

      > 靠右对齐

      ^ 居中对齐
      <宽度> 设定输出宽度
      <,> 数字的千位分隔符

      用于整数和浮点数
      <.精度> 浮点数小数部分的精度

      或字符串输出的长度
      <类型> 整数类型：b、c、d、o、x、X

      浮点数类型：e、E、f、%

    • <填充>、<对齐>、<宽度> 主要用于对显示格式的规范

    • <,>、<.精度>、<类型> 主要用于对数值本身的规范

      • <类型> 表示输出整数和浮点数类型的格式规则

      • 对于整数：

        • b：输出整数的二进制形式
        • c：输出整数对应 Unicode 字符
        • d：输出整数的十进制形式
        • o：输出整数的八进制形式
        • x：输出整数的小写十六进制形式
        • X：输出整数的大写十六进制形式

      • 对于浮点数：

        • e：输出对应的小写字母 e 的指数形式
        • E：输出对应的大写字母 E 的指数形式
        • f：输出标准浮点形式
        • %：输出浮点数的百分比形式

      • 范例：

  • 字符串处理方法

    • 常用字符串处理方法

      str.lower() 返回全部字符小写
      str.upper() 返回全部字符大写
      str.split([sep],[num]) 指定分隔符 sep 对字符串切片，省略 sep 默认为以所有的空字符分隔，
      如：空格、换行（ \n ）、制表符( \t )，

      str.join(iter) 将序列中的元素以指定的字符 iter 连接生成新的字符串
