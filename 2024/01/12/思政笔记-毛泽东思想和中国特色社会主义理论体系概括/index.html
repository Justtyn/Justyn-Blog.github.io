导论 马克思主义中国化时代化的历史进程与理论成果

一、马克思主义中国化现代化的内涵(习近平 - 二十大提出)

  • 时间线：新文化运动(1915) - 俄国十月革命(1917) - 五四运动(1919) - 党的成立(1921)

  • 马克思主义是关于自然界、社会、人类思维发展的一般规律的理论体系

  • 基本概念：

    • (两结合)：立足中国国情和时代特点，坚持把马克思主义基本原理同中国具体实际(国情)相结合、同中华民族传统文化相结合

  • ”马克思主义中国化“命题的正式提出：1938年 - 毛泽东 - 六届六中全会 - <论新阶段>

  • ”马克思主义中国化现代化“的提出：二十大 - 习近平提出

  • 必要性(为什么)：

    • 解决中国实际问题的[客观需要]
    • 马克思主义理论发展的[内在要求]

  • 时间点：

    • 1938年 毛泽东 六届六中全会 <论新阶段> 提出”马克思主义中国化“命题

    • 1840-1949 新民主主义革命、社会主义革命时期

    • 1949-1956 新民主主义社会 三大改造完成

    • 1956 进入社会主义社会 1978完成 社会主义建设时期 毛泽东运用马克思主义解决

    • 1978-今 从邓小平开始

二、马克思主义中国化时代化理论成果

  • 三大理论成果/三次历史性飞跃

    • 第一大：毛泽东思想

      • 关键词：新民主主义革命、社会主义革命、建设时期

    • 第二次：邓小平理论(邓小平)、三个代表重要思想(江泽民)、科学发展观(胡锦涛)

      • 统称为：中国特色社会主义理论体系(邓、三、科)

      • 关键词：改革开放

    • 第三次(新飞跃)：习近平新时代中国特色社会主义思想

      • 关键词：十八大以来(2012年)、中国梦

第一章 毛泽东思想及其历史地位

时间线：新民主主义革命(1919-1949) - 新民主主义社会(过渡时期、三大改造、社会主义改造/革命)(1949-1956) - 社会主义社会(1956-今)

第一节 毛泽东思想的形成和发展

一、毛泽东思想性发展的历史条件

  • 时代背景(革命与战争的时代)：19世纪末20世纪初，世界进入帝国主义和无产阶级革命时代(战争与革命成为时代主流)

    1840年 - 鸦片战争爆发 - 中国沦为半封建半殖民 - 英国敲开中国大门 - 在沿海城市设厂 - 5个通商口岸 - 农民形成无产阶级(工人阶级) - 工人无产阶级革命 - 帝国主义和无产阶级对立 - 帝国主义横行 - 战争与革命成为时代主流

  • 思想条件(理论基础)：十月革命给中国送来了马克斯列宁主义

    十月革命(1917年)：第一个社会主义国家苏联的建立-思想来源马克思主义-列宁学习形成马克思列宁主义-传入中国

  • 实践基础：革命和建设

    共产党领导人民进行革命和建设的成功实践是毛泽东思想形成和发展的实践基础

第二章 新民主主义革命理论(1840-1949)

第一节 新民主主义革命理论形成的依据

一、近代中国国情和中国革命的时代特征

时间背景：1840-1949

认清中国国情，是解决中国革命问题的基本前提、基本依据、总依据

国情 = 社会性质 = 半殖民地半封建社会(1840-1949) = 基本依据、前提、总依据

  • 近代中国国情(2023简答题)

    • 近代中国的社会性质(单选、多选题)

      • 近代中国最基本国情：鸦片战争后，中国逐渐成为半殖民地半封建社会(1840-1949)

        • 半殖民地是从国家的政治地位上看的，半封建是从社会经济结构上看的

        • 半殖民地是相对于完全殖民地而言的、半封建是相对于完全封建而言的

      • 原因：

        1. 帝国主义的侵略加速了封建社会自给自足的自然经济的解体，整个社会呈现出半封建性
        2. 帝国主义列强通过政治、经济和文化的侵略，使中国半殖民地化

    • 近代中国社会的主要矛盾(单选、多选题)

      • 主要矛盾：帝国主义和中华民族的矛盾(最主要矛盾)、封建主义和人民大众的矛盾

      • 近代中国革命的根本任务：反帝、反封建、反官僚

      • 国情决定主要矛盾、根本任务、革命的性质(资产阶级民主主义革命)

  • 近代中国革命的时代特征(单选、多选题)

    • 性质：近代和中国的社会性质和主要矛盾，决定了中国革命仍然是资产阶级民主革命，具有鲜明的时代特征

    • 旧民主主义革命(1840-1919)：鸦片战争至辛亥革命 - 农民阶级(义和团、太平天国)、地主阶级(洋务运动)、资产阶级(维新派改良、孙中山辛亥革命)

    • 以俄国十月革命(1917)的胜利为标志：中国资产阶级民主革命的时代背景发生了根本转换

      • 是新民主主义革命时代的背景
      • 俄国十月革命的胜利，开辟了世界无产阶级社会主义革命的新纪元，标志人类历史由资本主义向社会主义转变的进程
      • 性质：转变为属于新的资产阶级民主主义革命的范畴，属于世界无产阶级社会主义革命的一部分

    • 以“五四运动”(1919)为标志：中国无产阶级开始一独立的政治力量登上历史舞台

      • 革命阶段发生转变
      • 马克思主义逐步成为中国革命的指导思想，中国进入新民主主义革命阶段

    • 时代背景新民主主义革命时代背景转换的标志：俄国十月革命(1917)

    • 革命阶段转变：以五四运动(1919)为标志

    • 主观条件：五四运动

    • 客观条件：

      1. 革命失败
      2. 革命继续发展
      3. 一战的爆发

    • 中国革命两步走：

      • 第一步完成反帝、反封建的新民主主义革命任务
      • 第二步是完成社会主义革命任务

第二节 新民主主义革命的总路线和基本纲领

一、新民主主义革命的总路线

  • 提出：1939年毛泽东在<中国革命和中国共产党>一文中第一次提出“新民主主义革命”的科学概念

  • 总路线(指导路线)的内容：无产阶级领导(领导力量)，人民大众的(革命动力：工、农、小、民)，反对帝国主义、封建主义和官僚资本主义(敌人)的革命

  • 总路线指明了中国革命的对象、动力、领导力量

    • 新民主主义革命的对象

      • 首要问题：分清敌友
      • 帝国主义：中国革命的首要对象
      • 封建地主阶级：社会基础
      • 官僚资本主义：依靠帝国主义、勾结封建势力
      • 三座大山：帝国主义、封建主义、官僚资本主义

    • 新民主主义革命的动力

      • 革命动力(工、农、小、民)：无产阶级(最基本动力)、农民阶级(主力军)、城市小资产阶级、民族资产阶级
        • 贫雇农：最可靠的同盟军
        • 中农：可靠的同盟军
        • 城市小资产阶级(知识分子、小商人、手工业者、自由职业者)：可靠的同盟者
        • 民族资产阶级：带有两面性的阶级(革命的力量之一、革命关键时刻表现出明显动摇性)
      • 新民主主义革命实质上是党领导下的农民革命
      • 农民问题是中国革命的基本问题

    • 新民主主义革命的领导力

      • 无产阶级的领导权：是中国革命的中心问题、也是新民主主义革命理论的核心问题

      • 区分新旧民主主义革命的根本标志：革命的领导权是掌握在无产阶级还是资产阶级手中

      • 中国无产阶级称为领带力量的原因：

        • 共性优点：与先进生产方式相联系、没有私人占有的生产资料(资产阶级控制)、富于组织纪律性
        • 自身优点：
          • 中国无产阶级深受资本主义、封建势力、资产阶级的三重压迫，具有彻底的革命性
          • 无产阶级分布集中，有利于思想传播和强大力量的形成
          • 出身农民

      • 领导权的实现途径：建立统一战线、独立自主(基本策略)、建立武装革命(坚强支柱)、党(根本保证)

      • 实现领导权的关键：必须建立以工农联盟为基础的统一战线

    • 新民主主义革命的性质和前途

      • 中国革命的性质：资产阶级民主主义革命
      • 和旧民主主义革命的区别：所属世界革命范畴不同、革命的领导力量是无产阶级中国共产党、指导思想是马克思列宁主义、革命的前途是社会主义
      • 与社会主义革命范畴不同，仍属于资产阶级民主主义革命
      • 目标：建立无产阶级领导的各革命阶级的脸和专政
      • 联系：社会主义革命是无产阶级性质的革命、目标是消灭资本主义剥削和私有制

二、新民主主义的基本纲领

分为政治纲领、经济纲领、文化纲领

  • 政治纲领：推翻帝国主义、封建主义的统治，建立无产阶级领导的、以工农联盟为基础、各革命阶级联合专政的新民主主义共和国
    • 目标：建立新民主主义的共和国

  • 经济纲领

    • 没收官僚资产阶级的垄断资本归国家所有(双重性质)

  • 文化纲领：无产阶级领导的人民大众的反帝、反封建的文化，是民族的(反帝)科学的(反封建)大众的文化

      • 两个联盟
        • 工人阶级同农民阶级、广大知识分子及其他劳动者的联盟，主要是工农联盟
        • 工人阶级和非劳动人民的联盟，主要是与民族资产阶级的联盟
      • 实践经验
        • 建立巩固的工农联盟(最基础力量)

    • 统一战线和武装斗争是两个基本特点、武器
    • 统一战线是实行武装斗争的统一战线，武装斗争是统一战线的中心支柱
    • 党的组织是英勇战士

第三章 社会主义改造理论

1949-1956 - 社会主义革命(改造)

目的：让社会主义制度在中国确立

社会主义：不同于资本主义(物竞天择、适者生存、强调竞争关系)，社会主义强调人与人之间的互助合作、集中力量办大事、共同享受发展成果

1949-1956年 - 社会性质 = 新民主主义社会 = 不是一个独立的社会形态 = 过渡性的社会形态 = 新民主主义向社会主义过度 = 是属于社会主义体系的(经济(国营经济国民命脉)、政治(无产阶级领导)、国际条件(导向苏联社会主义阵营))

第一节 从新民主主义到社会主义的转变

一、新民主主义社会是一个过渡性的社会

  • 新民主主义社会的性质

    • 时间：中华人民共和国成立(1949) - 社会主义改造基本完成(1956)
    • 性质：我国的性质是新民主主义社会
      • 是我国从新民主主义到社会主义的过渡时期，是过渡性社会形态
      • 新民主主义社会不是一个独立的社会形态

  • 新民主主义社会的五种经济成分

    • 五个经济成分：社会主义性质的国营经济、半社会主义性质的合作社经济、农民和手工业者的个体经济、私人资本主义经济和国家资本主义经济
      • 国营经济：没收官僚资本而来，占有领导地位
      • 合作社经济：个体经济向社会主义集体经济过渡的形式
      • 个体经济：土地革命，是绝对经济
      • 私人资本主义经济：保护民族工商业而来
      • 国家资本主义经济：私人资本主义经济向社会主义国营经济过渡的形式
    • 主要经济成分(性质)：社会主义经济、个体经济、资本主义经济
    • 两个过渡经济：合作社经济(个体-社会主义集体，农村)、国家资本主义经济(私人-国营，城市)
    • 阶级领导力量：无产阶级

  • 新民主主义社会存在的阶级构成

    • 中国社会的阶级构成主要是：工人阶级、农民阶级和其他小资产阶级、民族资产阶级等基本的阶级力量
    • 工人阶级和资产阶级的矛盾逐渐成为我国社会的主要矛盾
    • 新民主主义社会属于社会主义体系的原因：社会主义的因素不论在经济还是政治上都已经居于领导地位

二、党在过渡时期的总路线及其理论依据

  • 走适合我国国情的社会主义工业化的历史必然性

    • 三点原因
      • 中国贫穷落后，需要从农业国变成先进的工业国
      • 当时的时代条件、国际环境的新特点(导向苏联)
      • 是国家独立和富强的必然要求和必然条件

  • 党在过渡时期的总路线

    • 主要内容概括为：”一化三改”
      • 一化：社会主义工业化
      • 三改：个体农业、手工业、资本主义工商业的社会主义改造(改变：生产资料私有制 - 公有制)
    • 这是一条社会主义建设和社会主义改造同时并举的路线

  • 党在过渡时期的总路线的理论依据

    • 马克思、恩格斯提出了资本主义社会向社会主义社会过渡的问题
    • 列宁进一步发展马克思、恩格斯的革命转变思想

第二节 社会主义改造道路和历史经验

一、适合中国特点的社会主义改造道路

  • 农业、手工业的社会主义改造

    • 农业的社会主义改造
      • 意义：个体经济转为集体经济
      • 积极引导农民组织起来、走互助合作道路
      • 遵循自愿互利、典型示范、国家帮助原则
      • 制定正确的阶级政策(消灭富农)
      • 方针：坚持积极领导、稳步前进方针，采取循序渐进步骤
        • 改造经历了三个发展阶段
          • 互助社(生产资料私人所有)、初级社(半社会主义性质，生产资料农民所有，但农具加入了合作社)、高级社(完全社会主义性质，生产资料公有)
    • 手工业的社会主义改造
      • 方针：党和政府采取积极领导、稳步前进方针
      • 步骤：
        • 办手工业供销小组(具有社会主义萌芽性质)
        • 办手工业供销合作社(半社会主义性质)
        • 建立手工业生产合作社(社会主义性质)
      • 方法：采取说服教育、示范和国家帮助、自愿参加

  • 资本主义工商业的社会主义改造

    • 意义：生产资料私有转向公有制(国营)

    • 方法：

      • 和平赎买方式改造
        • 含义和方式
          • 国家有偿的对私营企业改为国营企业
          • 将资本主义私有制改变为社会主义公有制
          • 具体方式不是一次性支付巨额赔偿金
          • 而是让资本家在一定年限内从企业经营所得中获得一部分利润(10-15年)
        • 原因
          • 有利于团结各民主党派和各界爱国民主人士
          • 有利于发挥民族资产阶级中大多数人的知识、才能、技术专长、管理经验
      • 采取从低级到高级的国家资本主义过渡形式
        • 含义：国家资本主义，就是在国家直接控制和支配下的资本主义经济
        • 过渡形式：
          • 第一步：采取经销、代销形式、“四马分肥”
            • 四马分肥：企业利润，按国家所得税、企业公积金、工人福利费、资方红利四方面进行分配
          • 第二步，实行个别企业的公私合营，国家投资入股，企业的生产资料由国家和资本家共同所有，企业利润分配仍为四马分肥
          • 第三步，实行全行业的公私合营
      • 把资本主义工商业者改造为自食其力的社会主义劳动者
        • 通过改造阶级成分的方式达到从整体上消除阶级的目的

二、社会主义改造的历史经验

  • 实践经验

    • 坚持社会主义工业化建设和社会主义改造同时并举
    • 采取积极引导、逐步过渡的方式
    • 用和平方法进行改造

第三节 社会主义基本制度在中国的确立

一、社会主义基本制度的确立及其理论根据

  • 确立：1956年底，社会主义改造基本完成
  • 标志：中国历史上长达数千年的阶级剥削制度的结束和社会主义基本制度的确立
    • 剥削制度的结束不代表没有剥削现象
  • 确立依据
    • 经济方面：生产资料公有制为基础的社会主义经济制度已经建立起来
    • 政治方面：1954年，第一届人民代表大会召开和中华人民共和国宪法的制定及颁布施行，成为我国社会主义民主政治建设的里程碑
    • 阶级方面：我国社会阶级关系发生了根本变化，工人阶级已经成为国家的领导阶级
  • 理论依据：列宁关于经济文化比较落后的国家可以走社会主义的理论依据

二、确立社会主义基本制度的重大意义

  • 内涵：社会主义基本制度的确立是中国历史上最深刻、最伟大的社会变革

  • 意义：

    • 经济意义：极大地促进了我国社会生产力的发展
    • 政治意义：使广大劳动人民真正成为国家的主人
    • 世界意义：是世界社会主义发展史上有一个历史性的伟大胜利
    • 理论意义：不仅再次证明了马克思列宁主义的真理性，且用独创性的理论原则和经验总结丰富和发展了科学社会主义理论
